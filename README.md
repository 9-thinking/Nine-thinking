# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/1bcd4b70-72de-42ff-bc08-487f5562a648

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Download Google Cloud CLI in https://cloud.google.com/sdk/docs/install-sdk?#windows to import your projects into cloud to get Gemini API key
3. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
4. Run the app:
   `npm run dev`
